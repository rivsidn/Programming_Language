`makefile` 中的四种赋值方式。



| 赋值符号 | 解释                                                         |
| -------- | ------------------------------------------------------------ |
| :=       | 直接赋值，等号右边的值赋值给左边                             |
| =        | 递归赋值，只有在用的时候，才将右侧的值展开赋值给左侧<br/>类似与宏展开 |
| ?=       | 变量还没赋值的情况下给变量赋值                               |
| +=       | 追加赋值                                                     |
